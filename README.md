# Vegetation Resistance to Climate Variability in Germany

This repository contains the data and code used to analyze vegetation response to climate variability across Germany.

## Data sources

- MODIS NDVI (MOD13Q1)
- ERA5-Land climate data

## Structure

- `data/`: processed dataset used for analysis
- `python/`: regression models and figure generation
- `gee/`: Google Earth Engine scripts
- `figures/`: final figures used in the paper

## Main findings

- Vegetation response is primarily driven by soil moisture rather than temperature
- Temperature influences vegetation indirectly through its effect on soil moisture
- Croplands are more sensitive to drought, forests are more resistant
